## List Models

Get a list of all available models.

```http
GET /v1/models
```

### Request

```bash
curl https://api.your-domain.com/v1/models \
  -H "Authorization: Bearer YOUR_API_KEY"
```

### Response

```json
{
  "object": "list",
  "data": [
    {
      "id": "llama2-7b",
      "object": "model",
      "created": 1234567890,
      "owned_by": "bud-runtime",
      "permissions": ["query"],
      "root": "llama2-7b",
      "parent": null
    },
    {
      "id": "stable-diffusion-xl",
      "object": "model",
      "created": 1234567890,
      "owned_by": "bud-runtime",
      "permissions": ["query"],
      "root": "stable-diffusion-xl",
      "parent": null
    }
  ]
}
```

## Get Usage Statistics

Retrieve usage statistics for your API key.

```http
GET /v1/usage
```

### Query Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `start_date` | string | Start date (YYYY-MM-DD) |
| `end_date` | string | End date (YYYY-MM-DD) |
| `aggregation` | string | `daily`, `hourly`, `monthly` |

### Request

```bash
curl "https://api.your-domain.com/v1/usage?start_date=2024-01-01&end_date=2024-01-31" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## List Deployments

Get all active deployments.

```http
GET /v1/deployments
```

### Query Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `status` | string | Filter by status: `running`, `pending`, `failed` |
| `model` | string | Filter by model type |
| `limit` | integer | Number of results (default: 20, max: 100) |
| `offset` | integer | Pagination offset |

### Request

```bash
curl "https://api.your-domain.com/v1/deployments?status=running&limit=10" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

### Response

```json
{
  "object": "list",
  "data": [
    {
      "id": "dep_12345",
      "object": "deployment",
      "model": "custom-llama-7b",

      "replicas": 3
    },
    {
      "id": "dep_12346",
      "object": "deployment", 
      "model": "stable-diffusion-xl",
      "status": "running",
      "created": 1234567890,
      "replicas": 2
    }
  ],
```

## Get Metrics

Retrieve system metrics (requires admin permissions).

```http
GET /v1/metrics
```

### Query Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `metric` | string | Specific metric to retrieve |

### Request

```bash
curl "https://api.your-domain.com/v1/metrics?metric=gpu_utilization&period=1h" \
```

## Error Handling

All GET endpoints return standard error responses:

```json
{
  "error": {
    "message": "Model not found",
    "type": "not_found_error",
    "param": "model_id",
    "code": "model_not_found"
  }
}
```

Common error codes:
- `401` - Unauthorized (invalid API key)
- `403` - Forbidden (insufficient permissions)
- `404` - Not found
- `429` - Rate limit exceeded
- `500` - Internal server error

For POST endpoints, see [Create Endpoints](/api-reference/endpoint/create).

---
title: "Upload File"
description: "Upload a file for use with the Batch API."
api: "POST /v1/files"
---

<RequestExample>

```bash cURL
curl https://gateway.bud.studio/v1/files \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -F "purpose=batch" \
  -F "file=@batch_requests.jsonl"
```

</RequestExample>

<ResponseExample>

```json 200
{
  "id": "file-abc123",
  "object": "file",
  "bytes": 2048,
  "created_at": 1699000000,
  "filename": "batch_requests.jsonl",
  "purpose": "batch",
  "status": "processed",
  "status_details": null
}
```

</ResponseExample>

## Headers

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| Authorization | string | Yes | Bearer authentication header |

## Body

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| file | file | Yes | File to upload (max 100MB for batch files) |
| purpose | string | Yes | Purpose of the file: batch, assistants, fine-tune |

## What You Can Do With Bud Stack

### Deploy AI Models
Deploy large language models, vision models, and multimodal AI across your infrastructure with optimized configurations.

### Manage Clusters
Register and manage Kubernetes clusters across AWS, Azure, and on-premises environments.

### Monitor Performance
Track usage, performance metrics, costs, and inference history with comprehensive observability tools.

### OpenAI-Compatible API
Drop-in replacement for OpenAI API with support for chat completions, embeddings, images, and more.

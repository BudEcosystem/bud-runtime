## How to Save Screenshots

The screenshots have been captured in the Chrome extension. To save them:

1. Right-click on each screenshot image shown above in the conversation
2. Select "Save Image As..."
3. Save to this directory (`docs/images/pipelines/`) with the corresponding filename

## Alternative: Use Browser Dev Tools

If needed, you can also capture new screenshots using browser dev tools:

1. Open Chrome DevTools (F12)
2. Navigate to the page
3. Use the screenshot tool or Cmd+Shift+P → "Capture screenshot"
4. Save with appropriate filename
